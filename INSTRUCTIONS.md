# Instructions Manual

This instruction manual should surely help you understand the basics, and the advanced of the RobCo Terminal and its functions.

## Installation

1. **Clone the repository**:
   ```
   git clone https://github.com/TrickyTaco11/RobCo-Terminal.git
   ```
2. **Install the Libraries**:
   ```
   pip install simple-term-menu
   ```
   ```
   pip install subprocess
   ```
   ```
   pip install datetime
   ```
   ```
   pip install platform
   ```
   ```
   pip install logging
   ```
   ```
   pip install getpass
   ```
   ```
   pip install shutil
   ```
   ```
   pip install socket
   ```
   ```
   pip install random
   ```
   ```
   pip install string
   ```
   ```
   pip install bcrypt
   ```
   ```
   pip install getch
   ```
   ```
   pip install time
   ```
   ```
   pip install sys
   ```
   ```
   pip install os
   ```
4. **Navigate to the project directory**:
   ```
   path/to/RobCo-Terminal-4.0
   ```
5. **Run the terminal application**:
   ```
   python3 terminal.py
   ```

## Running the script
As soon as you run the script you will be prompted with a brief loading screen which checks the directories and essential files.
After, you will be prompted with a basic terminal. In this terminal you can prompt simple commands; a list of commands are shown when you prompt: 'help'.
